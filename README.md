# template-vue-typescript

> A SAO template for Vue and TypeScript. 

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao vue-typescript my-project
```

### From git

```bash
sao gluons/template-vue-typescript my-project
```
